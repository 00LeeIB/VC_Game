# VC_Game

### 뱀서라이크 아레나 슈팅 로그라이크 게임
"VC_Game"은 뱀서류 스팀 게임 "Brotato"를 참고하여 개발된 간단한 게임입니다. 
C++ 및 Visual Studio를 사용하여 만들어졌으며, 학습 목적을 위해 만들었습니다.
